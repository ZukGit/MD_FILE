#  Tusahre
```
tushare简介; 
     Tushare是一个免费、开源的python财经数据接口包。主要实现对股票等金融数据从数据采集、
     清洗加工 到 数据存储的过程，能够为金融分析人员提供快速、整洁、和多样的便于分析的数据，
     为他们在数据获取方面极大地减轻工作量，使他们更加专注于策略和模型的研究与实现上。



官网： http://tushare.org/
```

函数1：  get_k_data   【获取K线数据集合】

Tusahre所能查询到的交易数据分类：
历史行情数据
复权历史数据
实时行情数据
历史分笔数据
实时报价数据
当日历史分笔
大盘指数列表
大单交易数据




get_k_data   【获取K线数据集合】
即能方便获取日周月的低频数据，也可以获取5、15、30和60分钟相对高频的数据
同时，上市以来的前后复权数据也能在一行代码中轻松获得，当然，您也可以选择不复权。


get_k_data(String code,String keyType,String autype ,  ）
参数说明：
code:  股票证券代码  为必须参数  其余为非必须有默认值的参数
keyType: K线类型  'D'日K线  'W'周K线  'M'月K线  '5'  '15' '30'  '60'
autype: 复权类型   qfq-前复权 hfq-后复权 None-不复权，默认为qfq
index： 是否为指数   默认为false  为true时表示 code 不是具体的股票而是综合指数
start：  开始查询日期   格式   YYYY-MM-DD 为空时默认取当前日期
end:     format：YYYY-MM-DD  为空时默认取当前日期


返回数据格式说明：
date:  日期和时间    低频数据时为：YYYY-MM-DD     高频数为：YYYY-MM-DD HH:MM
open:   股票开盘价
close:  股票收盘价
high:   股票当期最高价
low:    股票当期最低价
volume: 股票当期成交量   xxx 手股票 == volume*100 股股票的成交量
amount : 成交金额
code:   股票代码
price_change：价格变动
p_change：涨跌幅
ma5：5日均价
ma10：10日均价
ma20:20日均价
v_ma5:5日均量
v_ma10:10日均量
v_ma20:20日均量
turnover:换手率[注：指数无此项]


【注意】
使用Python进行数据分析时，通过print输出Dataframe中的数据，当Dataframe行数很多时，中间部分显示省略号
想通过print显示全部数据
通过pandas.set_option() 函数可以设置pandas相关的参数，从而改变默认参数。 
打印pandas数据时，默认是输出100行，多的话中间数据会输出省略号。
import   pandas
pandas.set_option('display.width', 1000)     # 设置字符显示宽度
pandas.set_option('display.max_rows', None)  # 设置显示最大行
